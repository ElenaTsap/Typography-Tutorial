Typography

color:

defines the color of the text

color name

You can use one of the 140+ color names.
color: red;

Hello World ;)

hex code

You can use hexadecimal color codes.
color: #05ffb0;

Hello World ;)

rgb value

You can use rgb() color codes:
  • the first value is for red
  • the second value is for green
  • the third value is for blue
Each of them can have a value between 0 and 255.
color: rgb(50, 115, 220);

Hello World ;)

rgba value

You can use rgba() color codes:
  • the first 3 values are for rgb
  • the 4th value is for the alpha channel and defines the opacity of the color
The alpha value can go from zero 0 (transparent) to one 1 (opaque).
color: rgba(0, 0, 0, 0.5);

Hello World ;)

hsl value

You can use hsl() color codes:
  • the first 3 values are for rgb
  • the 4th value is for the alpha channel and defines the opacity of the color
  • The alpha value can go from zero 0 (transparent) to one 1 (opaque). the first value is for hue and can go from 0 to 359
  • the second value is for saturation and go from 0% to 100%
  • the third value is for luminosity and go from 0% to 100%
color: hsl(14, 100%, 53%);

Hello World ;)

hsla value

You can use hsl()a color codes:
  • the first 3 values are for hsl
  • the 4th value is for the alpha channel and defines the opacity of the color

alpha value can go from zero 0 (transparent) to one 1 (opaque).

color: hsla(14, 100%, 53%, 0.6);

Hello World ;)

font-family:

defines the font family of the text

Internal Specific Font

When using multiple values, the font-family list of font families defines the priority in which the browser should choose the font family.
  • Browser tries Times New Roman first.
  • If its not possible to use tries Times.
  • Then it tries Serif
  • If the last one is also not possible to use, then it uses the default font.
font-family: 'Times New Roman', Times, serif;

Hello World ;)

External Specific Font with HTML link

Insert the font link in the <head>
<link href="https://fonts.googleapis.com/css2?family=Fredoka+One&display=swap" rel="stylesheet">

Hello World ;)

External Specific Font via CSS Import

External fonts can be used in HTML via @import url on the top of the css file.
@import url('https://fonts.googleapis.com/css2?family=Abril+Fatface&display=swap');

Hello World ;)

External Specific Fonts from the Project folder

External fonts can be used in HTML from the project folder as font files (like .ttf, .otf) After adding the font file to your project folder you have to name it in your CSS like this:
@font-face {font-family: "Eleni's Font", monospace;
src: url(./Assets/SourceCodePro-Light.ttf);}
Then you can call use it in your css like this:
font-family: "Eleni's Font", monospace;

Hello World ;)

font-size:

defines the font size of the text

Default value

The text will use the browser's default medium size.
font-size: medium;

Hello World

Pixel values

You can use pixel values.
font-size: 20px;

Hello World

em values

  • The value is relative to the parent's font-size.
  • As a result, the value will cascade if used on child elements.
font-size: 1.2em;
Parent container: 20px;
font-size: 1.2em;(24px)
font-size: 1.2em;(28.8px)
font-size: 1.2em;(36.56px)

rem values

  • The value is relative to the root element's font-size, which is the html element.
  • As a result, the value will not vary depending on the depth of the element in the HTML hierarchy, and will reamin context independent.
font-size: 1.2rem;
Parent container: 18px;
font-size: 1.2rem;(21.6px)
font-size: 1.2rem;(21.6px)
font-size: 1.2rem;(21.6px)

percentage values

  • The value is relative to the parent's font-size.
  • As a result, the value will cascade if used on child elements.
font-size: 90%;
Parent container: 18px;
Font-size: 90% = 16.2
Font-size: 90% = 14.58px
Font-size: 90% = 13.122px

relative keywords

the value is relative to the parent. The following are available:
  • larger
  • smaller
font-size: smaller;
Parent container: 18px;
Font-size: smaller
Font-size: smaller
Font-size: smaller

absolute keywords

You can use absolute keywords. The value is relative to the root element html. The following are available:
  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large
font-size: x-large;
html root element: 18px
Font-size: x-large
Font-size: x-large
Font-size: x-large

font weight:

defines the weight of the text.

normal

The text is in normal weight.

font-weight: normal;

Hello World

bold

The text becomes bold.

font-weight: bold;

Hello World

numeric values

You can use numeric values. They all correspond to a particular named weight:

  • 100 Thin
  • 200 Extra Light
  • 300 Light
  • 400 Normal
  • 500 Medium
  • 600 Semi Bold
  • 700 Bold
  • 800 Extra Bold
  • 900 Ultra Bold
If the font family doesn't provide the requested weight, it will use the closest available one.

font-weight: 600;

Hello World

relative keywords

You can use relative keywords: lighter or bolder. The browser will use the next available weight.

font-weight: lighter;

Hello World

oblique

font-awesome:

add beautiful icons

how to install:

Paste the script into your HTML's head.

<script> src="https://kit.fontawesome.com/dc3beb17c2.js" crossorigin="anonymous"></script>

now copy the HTML code and paste is in your files.

<i class="fas fa-cat"></i>

Edit the icon in your css, using font properties. You have to treat it as a font element!

.fas { font-size: 120px; color: white; margin: 10px; }

You can also edit the sizes from pre-set classes of font-awesome. The sizing is relative using "em" units. So they will always be relavant to the parent element.
