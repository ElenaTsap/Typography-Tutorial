Font-family

Default Value

Sets a specific font for the text
Syntax
font-family: "font (family) name"

Hello World ;)

Internal Specific Font {font-family:'Courier New', Courier, monospace;}

  • Browser tries Times New Roman first.
  • If its not possible to use tries Times.
  • Then it tries Serif
  • If the last one is also not possible to use, then it uses the default font.

External Specific Fonts from the Project folder {font-family: "Eleni's Font", monospace;}

External fonts can be used in HTML from the project folder as font files (like .ttf, .otf) After adding the font file to your project folder you have to name it in your CSS like this:
@font-face {
font-family: "Eleni's Font", monospace;
src: url(./Assets/SourceCodePro-Light.ttf);
}
